# Premium Live Music App

## 🚀 Features
- **Premium UI**: Black/gold Spotify-like design with gradients, animations
- **Music Library**: 20+ songs (Western, Hindi, Punjabi, Haryanvi)
  - Arijit Singh, Diljit Dosanjh, Guru Randhawa, Ruchika Jangid + classics
- **Live Radio**: 6 genres (Rock, Jazz, EDM, Classical, Pop, Hindi)
- **Tabs**: Home (sections), Live (grid), Search (live), Library
- **Player**: Global audio with just_audio, now playing bar, full screen
  - Play/pause/seek/background, LIVE indicators
- **Responsive**: Web/Android/iOS ready

## 🎵 Songs & Streams
- **Songs**: SoundHelix samples (add your MP3 in assets/audio/)
- **Live Streams**: Public radio URLs (real audio!)

## 🛠️ Setup
1. `flutter pub get`
2. Add MP3 files to `assets/audio/`
3. `flutter run` (Chrome/Android)
