# Premium Music App v2.0.0

## Features
- Local & streaming audio playback (just_audio)
- Playlist queue, skip, loop
- Downloads (Dio, path_provider)
- Background audio (audio_service)
- Real image covers (iTunes API)
- Search, library, live streams
- Material3 dark theme
- Downloads, offline support

## Run
```
flutter pub get
flutter run
```
